# pfp-thresholds
Computing thresholds from Prefix-free parsing

### Download

```console
git clone https://github.com/maxrossi91/pfp-thresholds
```

### Compile

```console
mkdir build
cd build; cmake ..
make
```

#### Debug

```console
mkdir build
cd build; cmake -DCMAKE_BUILD_TYPE=Debug ..
make
```
